Welcome to csglib¶

The csglib is a python package for CSG group. This package includes standalone python tools/pipelines for various analyses as well as standalone python modules that may be re-used in other projects. Users are also very welcomed to copy and re-use (cherry-pick) specific code snippets from csglib in their personal projects.

Tools¶

  • Genome-wide association studies (GWAS)
    • InflationFromFile
    • Harmonize
    • GWASCatalog
  • Pedigrees
    • TrioPuller
